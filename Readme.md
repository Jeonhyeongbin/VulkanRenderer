# 세팅
- vulkanSDK(https://vulkan.lunarg.com/sdk/home#windows)를 다운받아 프로젝트의 external폴더에 vulkanSDK폴더를 추가해주셔야합니다.
- vs2022버전의 윈도우에서만 빌드 및 실행이 가능합니다.
- compile.bat를 실행하여 쉐이더를 한번 컴파일해 줘야합니다.
